# Omar Portfolio

A simple, semantic, and responsive HTML/CSS portfolio. Built with VS Code and deployed via Netlify Pages.

## Pages
- `index.html` — main portfolio with hero, about, education, skills, cards, and contact sections
- `images.html` — external/local images with figure/figcaption


## Tech
- HTML5, CSS3
- Git & GitHub Pages

## Features
- Responsive design for mobile and desktop
- Semantic HTML structure
- Custom CSS cards for services, testimonials, and pricing
- Accessible navigation and tables
